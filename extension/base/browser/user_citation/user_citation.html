Type of citation:
Type of source:
Narrative sentence:
A narrative sentence, for the biography, that comes before the inline citation.
Label:
A label to describe how this source relates to the profile. e.g. "Death", "Death of son John", "1851 Census", or "Member of Parliament" (do not include the quotes).
Author:
The author(s) of the book or article.
Source title:
The title of the book or article or the name of the collection of records.
Source reference:
Where the source document may be found and where in the document. Possibly publisher, date published. etc.
Info from source:
Most relevant information in the source. e.g. "John Smith born 10 Apr 1832" (do not include the quotes).
URL:
Copy/paste the URL from the cited page or, if the site provides a permalink, use that instead. If the site may go away, consider generating a link on the Wayback Machine.
Link text: The link text for the hyperlink to the above URL. If omitted a visible URL will be used.
